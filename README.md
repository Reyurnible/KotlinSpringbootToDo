## About
SpringBoot in Kotlin and gradle project.

## Requirement

- Kotlin > 1.0.3
- Spring > 1.4.1
- Java > 1.8

## Create Project
http://start.spring.io/

## Build

```
// Build
$gradle build
// Run project
$ java -jar build/libs/${application name}.jar

// Build and run
$gradle bootRun
```

access to `http://localhost:8080/`
